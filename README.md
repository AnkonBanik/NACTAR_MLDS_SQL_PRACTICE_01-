# NACTAR_MLDS_SQL_PRACTICE_01-
SQL practice exercises designed to test core relational database skills, including Data Definition Language (DDL) for schema creation, Data Manipulation Language (DML) for updating and deleting records, and various Data Query Language (DQL) techniques like filtering, aggregation, and sorting.

Project Title: SQL Fundamentals: Employee Management System Practice

Description:
This repository contains a structured SQL practice module focused on real-world data management scenarios. It utilizes an employees dataset to demonstrate proficiency in:

Database Schema Design: Creating tables with primary keys and specific data types.

Data Retrieval: Using SELECT with WHERE, LIKE, and BETWEEN clauses.

Aggregation & Grouping: Utilizing COUNT, SUM, AVG, and HAVING to analyze departmental data.

Data Maintenance: Performing UPDATE and DELETE operations based on specific conditional logic.

Sorting & Constraints: Organizing output using ORDER BY and handling NULL values in commissions.
